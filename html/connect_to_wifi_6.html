Smartphone > Windows  (Nokia Lumia) > Connect to WiFi


Slide your finger left accross the display to access the main menu.

 1.Then Press Settings.













2.Then tap this slide button to turn the WiFi on.

If you did it right, your screen should look like the one below.














3. Select a network.












4.Then it will ask you to enter the password for the WiFi.







 






5.When you enter it click done and wait to connect.

6.If the password is correct now you should have internet connection.
