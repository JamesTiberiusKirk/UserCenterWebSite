What is the user centre?


The User Centre Group is a friendly and informal computer club for over 50s where groups of computer users and learners of varying skills meet to learn and help each other in a relaxed and friendly environment. The Centre is ran by a Committee of volunteer helpers and University representatives. There are three main things the User Centre offers and these are: access to computers and software, social interaction, and help. This help could be with using computers or other devices such as laptops, tablets or phones, using different software, using email or browsing the web.

The User Centre runs four drop in sessions a week, from Tuesday to Friday, that last two hours each from 10am to 12pm. Drop in sessions are run by volunteer helpers and are free. They provide members with help geared to their own particular needs through assistance from volunteer helpers of similar age. They offer a friendly environment where members can learn from one another with some time set aside for tea, coffee and a chat.
