Smartphone > Android > Connect to WiFi


1.Find your device’s settings by finding a shortcut in the applications

drawer.












2. Once in the settings page, at the top of the screen you should be able to find a WiFi option in the list. Tapping that should take you to the overall WiFi settings.














3. If the WiFi is on, you should be able to see a list of nearby WiFi networks including the network which you want to connect to.

To check if the wifi is on, look at the top of the screen for a switch.

The switch can be flipped by tapping.














4. Tap on the network which you want to connect to.

If the network is password protected, it will ask you for the password.

The password can often be found on the back of your router.

If the password was correct, you should now be connected to the WiFi.
